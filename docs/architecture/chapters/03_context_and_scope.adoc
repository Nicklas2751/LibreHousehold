=== User (Actor)

A user is a person who uses the app to manage their household tasks and expenses, either alone or together with other users.

=== Administrator (Actor)

An administrator is a person who hosts an instance of the app and manages its configuration, users, and data.

=== E-Mail Server (External System)

The E-Mail Server is an external system that handles sending e-mails for the app, such as notifications and invitations.
